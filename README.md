# Bike Sharing Revenue Prediction — Machine Learning

Predicted revenue impact of converting casual users to registered 
subscribers for a bike-sharing company using machine learning.

## Business Problem
Casual users (2-day plan) generate unstable income. This project 
predicts how revenue changes when casual users switch to registered 
(1-month) subscriptions — helping the company grow stable revenue.

## What I Did
- Cleaned and preprocessed the Bike Sharing dataset
- Removed outliers using IQR method
- Engineered features: season, weather, user behaviour
- Built and compared Random Forest vs Decision Tree models
- Evaluated using R² score and cross-validation

## Results
- Random Forest outperformed Decision Tree
- Identified weather and season as top revenue predictors
- Model helps business target the right casual users to convert
